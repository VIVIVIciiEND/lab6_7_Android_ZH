## *Лабораторная работа №6-7. Взаимодействие с пользовательским*
# *Цель работы*
Познакомиться с понятием состояния (state) в Android-приложениях и научиться
использовать его в интерфейсе с помощью Jetpack Compose. В результате
лабораторной работы будет создано приложение калькулятор чаевых (Tip Time),
которое автоматически рассчитывает сумму чаевых. Изучить автоматизированные
тесты в Android.
# *Функциональность*
Приложение рассчитывает чаевые для ресторана. Пользователь вводит сумму счёта и процент чаевых, а программа показывает итоговую сумму чаевых. Есть возможность округлить чаевые вверх. В приложении использован удобный интерфейс с иконками и полями ввода, а результат отображается в формате валюты.
